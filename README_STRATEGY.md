# HaboRisk - 戦略ドキュメント

## 1. プロダクト概要 (逆引きPRD)
- **ターゲットユーザー**: 日本の中小企業の経営者・人事担当者（従業員10〜300名規模、人手不足や後継者問題に直面している企業）
- **解決する課題**: 自社の人手不足リスクを定量的に把握できない。どの部署・どの人材が最もリスクが高いか不明で、対策の優先順位がつけられない
- **ソリューション**: 従業員データを入力すると、5つの診断軸で人手不足リスクを0〜100でスコア化。部署別分析・キーパーソン特定・12ヶ月の人員予測・業界ベンチマーク比較を提供
- **コア機能**:
  - 5軸リスク診断（年齢構成25%・キーパーソン依存25%・離職率20%・季節変動15%・事業承継15%の加重スコアリング）
  - キーパーソン分析（スキル属人性の可視化、単一担当スキルの特定）
  - 12ヶ月先までの人員予測（離職率ベースの自然減予測、定年退職調整付き）
  - 部署別リスクヒートマップ
  - リスクスコアに基づく自動アクションプラン提案（優先度: urgent/high/medium/low）
  - 業界ベンチマーク比較（業種別平均離職率・平均年齢との比較）
  - データのエクスポート/インポート機能

## 2. Stripeマネタイズ戦略
- **現状**: Stripe未実装（localStorageベースのオフラインアプリ）
- **無料プラン制限**: 従業員登録10名まで、診断結果の保存は最新1件のみ、アクションプラン上位3件まで表示
- **プレミアム機能**:
  - 従業員数無制限登録
  - 診断履歴の保存・時系列比較（リスクスコアの改善トレンド追跡）
  - アクションプラン全件表示＋進捗管理
  - 12ヶ月人員予測グラフの詳細表示
  - PDFレポート出力（経営会議・銀行提出用）
- **推奨価格**: ¥1,980/月（年額¥19,800）
- **ペイウォール配置**: 従業員11人目の登録時、アクションプラン4件目以降の表示時、人員予測グラフの3ヶ月目以降

## 3. リテンション施策
- **メール通知トリガー**:
  - 四半期ごとの「リスク再診断リマインド」
  - 登録従業員の想定退職日（定年）が近づいた際のアラート
  - アクションプランの未着手タスクの月次リマインダー
- **データ蓄積フック**: 従業員データ・スキルマップ・診断履歴が蓄積されるほど、予測精度とベンチマーク比較の価値が向上。データのエクスポート/インポート機能がデータポータビリティを担保しつつ、分析機能で離脱を防止
- **習慣形成**: 四半期ごとのリスク再診断サイクルの確立。新入社員・退職者が発生するたびにデータ更新するフローを定着化

## 4. 不足機能 (Quick Wins)
1. **採用コストシミュレーター** - リスクスコアに基づき「このまま対策しない場合の採用コスト」を試算。経営者への危機感醸成と有料化の動機付け
2. **スキルマトリクス可視化** - 従業員×スキルのマトリクス表を表示し、属人性の高い業務を一覧化。キーパーソンリスクの具体的な対策立案を支援
3. **診断結果PDFレポート** - 経営会議や金融機関への提出用にレポートをPDF化。BtoB利用で必須の機能
